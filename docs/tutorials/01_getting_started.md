# Getting Started

Welcome to SAFARI — an AI-powered wildlife detection platform.

## What You'll Learn

In this tutorial, you'll learn how to:
- Create a project and upload datasets
- Label images and videos
- Train detection and classification models
- Run inference on new media

## Requirements

| Requirement | Details |
|-------------|---------|
| Browser | Chrome, Safari, or Firefox |
| Account | SAFARI login credentials |
| Data | Images or videos of wildlife |
