# Image Labeling

> Learn to annotate images with bounding boxes, manage classes, and edit masks.

---

## Entering the Editor

To start labeling images in a dataset:

1. Open your project from the dashboard
2. Click the **dataset card** — this opens the labeling editor directly

> To manage the dataset's files instead (upload, delete, view statistics), click <span class="icon-btn"><i data-lucide="settings-2"></i> Manage</span> on the card.

![Labeling studio card](screenshots/Labeling_studio_card.png)

---

## Editor Layout

The image editor uses a three-panel layout:

![Image editor layout](screenshots/Image_editor.png)

| Panel | Contents |
|-------|----------|
| **Left Sidebar** | Image thumbnails, progress bar ("X of Y labeled"), dataset statistics |
| **Center Canvas** | The image with all annotations overlaid. Supports zoom (scroll wheel) and pan (Shift+drag) |
| **Right Sidebar** | Tool buttons, class list, annotations for the current image |

The current image position is shown at the top of the left sidebar (e.g., "3/47"), along with a green progress bar showing how many images have been labeled.

---

## Drawing Bounding Boxes

1. Select the <span class="icon-btn solid-blue"><i data-lucide="square"></i> Draw</span> tool by pressing **R** or clicking the rectangle icon in the toolbar
2. **Click and drag** on the image to draw a bounding box
3. The box is automatically assigned to the currently selected class
4. Release the mouse — the annotation appears immediately on the canvas and in the right sidebar's annotation list

> **Tip**: Select your target class *before* drawing. Use number keys **1–9** to quickly switch between classes.

---

## Selecting & Editing

Switch to the <span class="icon-btn solid-blue"><i data-lucide="mouse-pointer-2"></i> Select</span> tool by pressing **V** or clicking the pointer icon.

- **Select** an annotation by clicking it on the canvas or in the annotations list
- **Resize** by dragging the corner handles of the selected box
- **Move** by dragging the interior of the selected box
- **Change class** — click the <span class="icon-btn icon-only"><i data-lucide="pencil"></i></span> icon next to the annotation in the right sidebar, or **right-click** the annotation on the canvas to open the context menu
- **Delete** — press **Delete** or click the <span class="icon-btn icon-only"><i data-lucide="trash-2"></i></span> icon in the annotations list

### Right-Click Context Menu

Right-clicking an annotation on the canvas opens a context menu with:

- **Assign Class** — a quick list of all project classes with color indicators, click to reassign
- **More Actions** — a submenu with:
  - **Use as Project Thumbnail** — sets the current annotated view as the project's thumbnail image
  - **Use as Dataset Thumbnail** — sets it as the dataset's thumbnail image

## Class Management

Classes are the categories you assign to annotations (e.g., *"Lince"*, *"Veado"*, *"Javali"*).

### Adding a Class

1. In the right sidebar, type the class name in the **"New class..."** input
2. Press **Enter** or click the <span class="icon-btn icon-only outline-green"><i data-lucide="plus"></i></span> button

### Selecting the Active Class

- Click a class in the list to make it active — all new annotations will use this class
- Use number keys **1–9** to quickly select a class by its position

### Color Coding

Each class is assigned a unique color using HSL rotation, ensuring visual distinction on the canvas. The same color appears on bounding box edges, labels, and the class list.

---

## Editing Masks

Some annotations include polygon masks generated by SAM3 autolabeling. You can refine these masks in the editor.

1. Select the <span class="icon-btn solid-blue"><i data-lucide="pentagon"></i> Mask Edit</span> tool by pressing **C** or clicking the pentagon icon
2. The selected annotation's mask polygon becomes editable with draggable vertices
3. **Drag a vertex** to reshape the polygon
4. **Click an edge** to add a new vertex
5. Use the **"Delete Mask"** button to remove the polygon mask while keeping the bounding box

> Mask editing is only available on annotations that have SAM3 mask data. Annotations without masks will only show the bounding box.

## Navigation

### Browsing Images

- Click any **thumbnail** in the left sidebar to jump to that image
- Press **A** for the previous image, **D** for the next image
- The current position indicator (e.g., "12/47") updates as you navigate

### Progress Tracking

- Images with at least one annotation show a green checkmark on their thumbnail
- The progress bar at the top of the left sidebar shows the labeled percentage
- Click the <span class="icon-btn icon-only"><i data-lucide="bar-chart-2"></i></span> statistics icon to view a breakdown of labeled vs. unlabeled images

### Multi-Select

- **Long-press** a thumbnail to toggle selection for batch operations
- **Shift+click** to select a range of images
- Selected images can be deleted in bulk via the selection action bar

---

## Autosave

All changes are saved automatically — there is no save button.

- Annotations are persisted when you navigate between images or close the editor
- A save indicator appears briefly when changes are being written
- If you need to undo, use the delete function on individual annotations
